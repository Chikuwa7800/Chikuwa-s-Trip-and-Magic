## Overview
<p>This data pack looks set to be quite a big one. It's designed to make travel and magic more enjoyable.</p>
<p>Travel-related features include new weapons, food, structures, bosses, and new mobs. Magic-related features include fire, ice, water, bolt, nature, and void spells, as well as magic staffs, spell books, and magical weapons.</p>
